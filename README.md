## Descrição

Este repositório organiza o trabalho acadêmico da disciplina de **Análise e Modelagem de Sistemas** com base no caso **BergaWorks Money**. O projeto reúne os requisitos levantados a partir da entrevista, os diagramas UML e uma implementação didática em Python derivada do modelo conceitual.

## Conteúdo

- **docs/requisitos**: requisitos funcionais e não funcionais.
- **docs/casos_de_uso**: resumo dos atores e casos de uso modelados.
- **docs/descrição_de_classes**: explicação das classes do domínio.
- **docs/img**: diagramas em PNG e em PlantUML.
- **docs/caderno_respostas**: versão final do trabalho em DOCX e PDF.
- **src**: implementação didática em Python para ilustrar a conversão dos requisitos em classes.

## Como executar

```bash
cd src
python main.py
```
