# Sistema BergaWorks Money

Projeto acadêmico de análise de requisitos, modelagem UML e implementação didática em Python, desenvolvido com base no estudo de caso **BergaWorks Money**.

![UML](img/Banner%20Github%20-%20ADS%20v2.png)

## Descrição

Este repositório reúne a documentação do trabalho da disciplina de Análise e Modelagem de Sistemas. O conteúdo foi organizado a partir do levantamento de requisitos do caso proposto, incluindo requisitos funcionais e não funcionais, diagramas UML e uma implementação didática em Python derivada do modelo conceitual.
