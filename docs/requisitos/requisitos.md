# Requisitos do sistema BergaWorks Money

## Requisitos funcionais

1. O sistema deve permitir que a funcionária PCD e a gestora solicitem a abertura da porta de entrada por comando de voz.
2. O sistema deve permitir a abertura da porta de entrada e do setor do usuário por celular, usando QR Code ou aproximação.
3. O sistema deve permitir ligar e desligar as luzes por comando de voz.
4. O sistema deve permitir ligar e desligar o ar-condicionado por comando de voz.
5. O sistema deve restringir a abertura das salas do RH e da gerência conforme o perfil de acesso do usuário.
6. O sistema deve enviar notificação à gestora quando houver movimentação fora do horário comercial.

## Requisitos não funcionais

1. O sistema deve ser implantado em servidor local (on premise), sem depender de servidor pronto em nuvem.
2. As salas críticas devem contar com câmeras para reconhecimento e monitoramento.
3. O servidor deve usar internet apenas para envio de notificações, preservando a segurança dos dados.
4. O sistema deve manter disponibilidade operacional no horário comercial definido pela empresa.
5. O acesso por celular deve operar com suporte à autenticação por QR Code e aproximação.
6. A solução deve integrar portas automatizadas, controle de ambiente e monitoramento em uma única infraestrutura local.
